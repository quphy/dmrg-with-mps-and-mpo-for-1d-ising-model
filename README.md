While learning DMRG, I was captivated by the genious design of MPS and MPO within it, and deeply impressed by the magic of tensor network methods represented by DMRG. Therefore, I decided to reproduce the DMRG implementation of MPO and MPS without using other existing tensor network libraries, such as ITensor. I chose the one-dimensional quantum Ising model because its Hamiltonian is simple enough, and the corresponding MPO is known.

The previous cpp code had a bug. I've now used the more efficient NumPy to handle tensor operations and rewritten the DMRG code in Python, including finite DMRG implementations for 1D ising models (PBC, OBC, and infinite DMRG implementations).
